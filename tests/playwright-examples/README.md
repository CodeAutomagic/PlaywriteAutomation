# Playwright Examples

This folder contains small, focused Playwright test examples demonstrating common APIs:

- `navigation.spec.js` — navigation and history
- `locators.spec.js` — locator patterns
- `frames.spec.js` — iframe interactions
- `popups-dialogs.spec.js` — popup & dialog handling
- `downloads-uploads.spec.js` — file download/upload examples
- `network-interception.spec.js` — route interception and mocking
- `tracing-screenshots.spec.js` — tracing and screenshots

Run the examples with:

```
npm run test:examples
```
